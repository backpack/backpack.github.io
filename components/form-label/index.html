Form label

Label text is used to inform users as to what information is requested for a text field. Every text field should have a label.

Edit page

Usage advice

DoDon't
Always use labels with form elements to provide context to the user.
Rely on placeholders alone.
Use the fieldset component which combines labels, fields and validation messages where possible.

Default

Disabled

Required

Invalid

Implementation

bpk-component-label

Backpack label component.

Installation

npm install bpk-component-label --save-dev

Usage

import React from 'react';
import BpkLabel from 'bpk-component-label';

export default () => (
  <BpkLabel htmlFor="origin">Origin</BpkLabel>
)
